## 核心哲学：追本溯源

在龍魂系统里，我们相信三个不可动摇的事实：

1. **任何数字行为都会留下行为指纹。**  代码可以复制，行为模式不能。
2. **真正的安全不是加密，而是可追溯。**  知道「谁、在什么时间、以什么方式、对什么做了什么」，比任何防火墙都可靠。
3. **主权不是法律给的，是技术赋能的。** 人民不需要等待授权，人民可以自己构建规则。

这就是**追本溯源**——不是事后追查，而是让每一次交互从出生那一刻起就自带血统证明。

## 七因子行为密码学

龍魂系统的核心引擎。详见 [`docs/seven-factor-cryptography.md`](docs/seven-factor-cryptography.md)。

> 七个因子，七个维度，构成数字身份的「行为 DNA」。

| 因子 | 符号 | 含义 |
|---|---|---|
| 操作习惯 | H₁ | 打字节奏、快捷键偏好、输入模式 |
| 代码风格 | H₂ | 命名习惯、缩进、注释风格、错误处理 |
| 决策路径 | H₃ | 面对选择时的优先顺序与分支模式 |
| 时间模式 | H₄ | 工作节律、响应延迟、周期性行为 |
| 空间模式 | H₅ | 设备指纹、网络拓扑、地理位置指纹 |
| 交互模式 | H₆ | 对话风格、确认习惯、反馈方式 |
| 演化模式 | H₇ | 长期迭代中的偏好变化与稳定性 |

综合公式：

```
Σ(C) = Σᵢ₌₁⁷ wᵢ · Hᵢ(C)
```

其中 `C` 为某个行为主体，`wᵢ` 为因子权重，由场景动态调整。

## 为什么不是专利？

龍魂系统不申请专利，原因有五：

1. **专利法有国界，龍魂没有。** 国与国之间的游戏规则保护不了民间创新。
2. **公开换保护是亏本买卖。** 申请专利要把核心算法写成说明书公开。
3. **20 年期限太短。** 龍魂要传世，不要 20 年保护期。
4. **跨国执行成本极高。** 个人打不起国际官司。
5. **法律保护不如技术防伪。** DNA 追溯 + 行为密码学，比任何专利证书都牢靠。

我们的保护方式是：**时间领先 + 文化壁垒 + 社区生态 + 人格主权**。
