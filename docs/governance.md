### 日常决策

由 Kimi Code CLI / 龍芯北辰主控执行，操作自动留痕。

### 重大决策

需满足：

1. 有 DNA 追溯码
2. 有书面理由
3. 有反对意见记录
4. 有最终确认码

## 争议解决

1. **证据优先。** 任何争议以 DNA 追溯链和七因子分析为准。
2. **本地举证。** 举证责任在提出方，证据必须本地可验证。
3. **社区仲裁。** 重大争议提交社区，按君子协议投票。
4. **不可诉诸外部法庭。** 龍魂系统内部争议优先用内部机制解决，但不排斥用户依法维权。
