# 行为准则 · Code of Conduct

> DNA: `#龍芯⚡️2026-07-03-ONGHUN-CODE-OF-CONDUCT-v1.0`

---

## 我们的承诺

我们致力于建立一个尊重、透明、建设性的社区。

## 我们的标准

**鼓励的行为：**

- 用证据说话
- 尊重不同背景的人
- 接受批评，改进自己
- 帮助新人理解龍魂理念
- 为人民利益发声

**禁止的行为：**

- 人身攻击、网络审判
- 散布未经证实的谣言
- 利用龍魂技术监控他人
- 商业投机、带货引流
- 煽动对立、制造仇恨

## 执行

违反行为准则者，社区管理员有权：

1. 警告
2. 标记
3. 限制参与
4. 永久移除

所有处理都需留下 DNA 追溯记录。
