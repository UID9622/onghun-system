## 欢迎

感谢你对龍魂系统感兴趣。我们不是普通开源项目，我们在构建一套数字主权基础设施。

在贡献之前，请确保你认同：

1. **世界老百姓最高**（零号协议）
2. **技术为人民服务**
3. **数据主权归用户自己**
4. **非商业、透明、留痕**

---

## 如何贡献

### 1. 提交 Issue

发现 bug、有改进建议、想讨论设计，请提交 Issue。

请包含：
- 问题描述
- 复现步骤
- 期望结果
- 相关 DNA 追溯码（如有）

### 2. 提交 Pull Request

- Fork 本仓库
- 在分支上修改
- 确保所有修改都有 DNA 追溯码
- 提交 PR 时说明改动原因

### 3. 文档贡献

文档是本系统最重要的部分之一。欢迎补充：
- 翻译
- 示例
- 教程
- 问题排查

---

## 代码规范

- 中文优先，英文兜底
- 每个文件顶部必须包含 DNA 追溯码
- 所有操作必须可审计
- 不引入外部商业依赖
- 不收集用户隐私数据
